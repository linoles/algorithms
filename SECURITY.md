# Security Policy

## Supported Versions

Этот проект находится в стадии активного обучения и не имеет версионных релизов. Актуальная версия кода — всегда последний коммит в ветке `main`.

| Версия | Поддержка |
|--------|-----------|
| `main` (последний коммит) | :white_check_mark: Полная |
| Предыдущие коммиты | :x: Не поддерживаются |

> **Рекомендация:** Если вы используете код из этого репозитория в своём проекте, регулярно выполняйте `git pull`, чтобы получать обновления.
